# e-norm

Mock-up d'une application de gestion. Dans un premier temps : gestion d'une BDD.

## Première étape : création d'un compte par une interface web

* BDD : `Postgres`
* API : `Node.js`
  * BDD-driver : `pg`
  * Logging : `Bunyan`
  * Authentification :  `Passport`
* Web app : `React` avec `Next.js`
